Configuração de Webhook para o Aplicativo Instagram

Este aplicativo permite que você configure webhooks para receber notificações em tempo real do Instagram.

Como funciona:

1. O aplicativo deve estar no modo "Publicado".
2. A URL do webhook deve ser configurada no painel de controle do Facebook Developer.
3. O webhook será ativado ao receber eventos do Instagram, como atualizações de perfil ou novos posts.
4. Toda a comunicação será feita de forma segura, e os dados nunca serão compartilhados com terceiros.

Exemplo de Configuração de URL:

- URL de Webhook: https://seu-dominio.com/webhook
- Token de Verificação: seu_token_de_verificacao

Se você tiver dúvidas sobre a configuração do webhook, entre em contato com nosso suporte.
